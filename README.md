# 📊 Stock Market Data Analyzer (HCLTECH)

🚀 A data analysis and visualization project built using Python and Streamlit to analyze stock trends, risk, and performance.


## 📌 Project Overview
This project is a Stock Market Data Analyzer developed using Python. It focuses on analyzing historical stock data of HCLTECH (NSE) and presenting insights through an interactive dashboard built with Streamlit.

The system performs data preprocessing, statistical analysis, and visualization to help understand stock trends, risk, and performance.

---

## 🎯 Objectives
- Analyze stock price movements using real-time techniques
- Apply data science concepts using Pandas and NumPy
- Visualize insights through an interactive dashboard
- Identify trends, volatility, and trading signals

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Plotly
- Streamlit

---

## 📂 Dataset
The dataset used is HCLTECH stock data (NSE), containing:
- Date
- Open Price
- High Price
- Low Price
- Close Price
- Volume

---

## ⚙️ Features Implemented

### 📊 Data Analysis
- Moving Averages (MA10, MA50)
- Daily Returns
- Volatility Analysis

### 📈 Technical Indicators
- RSI (Relative Strength Index)
- Bollinger Bands
- Trend Detection

### 📉 Risk Analysis
- Drawdown Calculation
- Return Distribution

### 📅 Time-Based Grouping
- Daily, Weekly, Monthly, Yearly aggregation

### 📢 Trading Signals
- Buy/Sell signals based on moving averages

### 🌐 Interactive Dashboard
- Built using Streamlit
- Date filtering
- Dynamic charts and metrics

---

## 🖥️ How to Run the Project

1. Install dependencies:
pip install -r requirements.txt
2. Run the application:
streamlit run app.py
3. Open in browser:
http://localhost:8501
